Документация к сайту прогноза погоды

1. Описание проекта

Этот веб-сайт предоставляет пользователям актуальную информацию о погоде в выбранных городах. Он использует API OpenWeatherMap для получения данных о погоде в реальном времени.

2. Технологии

  • Язык программирования: Python
  • Фреймворк: Django
  • Стили: Bootstrap
  • База данных: SQLite (по умолчанию)

3. Функционал

  • Добавление города и получение погоды
  • Автоматическое обновление данных
  • Ограничение списка городов (хранится только 5 последних)
  • Страница с подробным прогнозом

4. API для получения данных о погоде

Используемый API: OpenWeatherMap

Пример запроса:

https://api.openweathermap.org/data/2.5/weather?q=Minsk&units=metric&appid=YOUR_API_KEY

5. Возможные улучшения

  • Добавить автоматическое обновление погоды через AJAX
  • Улучшить дизайн и адаптивность
  • Добавить поиск города в реальном времени
  • Использовать кэширование запросов
